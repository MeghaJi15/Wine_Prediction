The Wine Quality Prediction system is a machine learning application designed to evaluate the quality of red wine based on its physicochemical properties. Using advanced data analysis and a trained classification model, the system predicts whether a wine falls into the premium or low-quality category.

 How It Works
The model analyzes important chemical attributes such as:
Acidity levels
Alcohol content
Sulphates
pH value
Density
Residual sugar
These features are processed using data preprocessing techniques like standard scaling, and a Random Forest Classifier is trained to identify patterns associated with high-quality wines.
To improve efficiency and accuracy, only the most influential features (based on feature importance analysis) are selected for final prediction.

 Machine Learning Approach:
Data preprocessing and normalization
Binary classification (Premium ≥ 7 quality score)
Feature importance ranking
Model training using Random Forest
Accuracy evaluation on test data

 Objective
The goal of this project is to demonstrate how machine learning can:
Assist in quality assessment
Reduce manual evaluation effort
Provide fast and reliable predictions
Help wineries and distributors make data-driven decisions

Applications
Wine manufacturing quality control
Data science and ML portfolio project
